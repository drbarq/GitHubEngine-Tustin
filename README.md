# GitHub Engine

> Web application that makes searching GitHub a breeze

## Starting Project

> Available Scripts

**Start the Redis cache**

`cd server`

`redis-server`

> check redis is connected: should recieve PONG back

`redis-cli ping`

**Start the server**

`nodemon server.js`

**Start GitHub Engine**

`cd ..`

`cd githubengine`

`npm start`

**Run tests for GitHub Engine**

`npm test`
